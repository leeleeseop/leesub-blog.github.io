인터야드(interyard)팀 프로젝트

요약

  • interYard는 온라인 종합 플랫폼에서 중고장터, 댓글 개발 담당을 맡게 되었습니다.

역할

  • 프로젝트 팀 인원 7명
  • 역할: 부PM (PM보조)
  • 담당: 중고장터 & 댓글

프로젝트 기간

  • 프로젝트 진행 기간 (2024.7.19 ~ 2024.8.6)

도구 및 기술

  • DBMS: Oracle11g xe
  • 개발언어: JAVA(JDK17), HTML, CSS
  • 라이브러리: jQuery, Bootstrap
  • 개발도구: Oracle SQL Developer
  • 템플릿 엔진: JSP
