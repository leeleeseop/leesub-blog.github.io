세덱스(SedEx) 팀 프로젝트

요약

  • 세덱스(SedEx)라는 스포츠 용품 쇼핑몰에서 장바구니 백엔드 개발을 담당하게 되었습니다.

    • 역할

  • 프로젝트 팀 인원 5명
  • 역할: 팀장
  • 담당: 장바구니

    • 프로젝트 기간

  • 프로젝트 진행 기간 (2024.05.17 ~ 2024.05.28)

    • 도구 및 기술

  • DBMS: Oracle11g xe
  • Java (JDK17)
  • 개발도구: Oracle SQL Developer

    • 문서작업

    프로젝트

    0.장바구니 메뉴

  • 장바구니의 메뉴를 선택해서 들어갑니다.
    • 메뉴출력
  • System.out.print("| 1.리스트 | 2.수정 | 3.삭제 | 4.담기 | 0.이전메뉴 |");
    • 1. 리스트: 사용자가 장바구니에 추가한 상품을 볼수있습니다. 2. 수정: 장바구니에 있는 상품의 수량을 수정할 수 있습니다. 3. 삭제: 장바구니에 있는 상품을 삭제 할 수 있습니다. 4. 담기: 장바구니에서 상품을 담을 수 있습니다.
