세덱스(SedEx) 팀 프로젝트

요약

  • 세덱스(SedEx)라는 스포츠 용품 쇼핑몰에서 장바구니 백엔드 개발을 담당하게 되었습니다.

    • 역할

  • 프로젝트 팀 인원 5명
  • 역할: 팀장
  • 담당: 장바구니

    • 프로젝트 기간

  • 프로젝트 진행 기간 (2024.05.17 ~ 2024.05.28)

    • 도구 및 기술

  • DBMS: Oracle11g xe
  • Java (JDK17)
  • 개발도구: Oracle SQL Developer

    • 문서작업

    요구사항 정의서

    테이블 정의서

    Flow Diagram

    프로젝트

    0.장바구니 메뉴

  • 장바구니의 메뉴를 선택해서 들어갑니다.
    • 메뉴출력
  • System.out.print("| 1.리스트 | 2.수정 | 3.삭제 | 4.담기 | 0.이전메뉴 |");
    1. 리스트: 사용자가 장바구니에 추가한 상품을 볼수있습니다.
    2. 수정: 장바구니에 있는 상품의 수량을 수정할 수 있습니다.
    3. 삭제: 장바구니에 있는 상품을 삭제 할 수 있습니다.
    4. 담기: 장바구니에서 상품을 담을 수 있습니다.

    1. 장바구니 리스트

  • 장바구니의 리스트로 장바구니 번호, 회원번호, 등 데이터를 보여주며 장바구나의 총금액 과 구매 여부를 묻도록 개발했습니다.

    • 2. 장바구니 수정

  • 장바구니 수정에서는 번호를 입력한 후, 수량을 수정할 수 있습니다. 수정 후 수정완료와 수정취소를 입력하면 장바구니 메뉴로 돌아갑니다.

    • 3. 장바구니 삭제

  • 장바구니 삭제에서는 “ 선택삭제” 와 “전체삭제”를 할수있도록 만들었습니다.
    • 선택삭제: 장바구니 번호를 입력하면 해당번호 만 삭제 할 수 있습니다.
    전체삭제: 장바구니 번호를 입력하지 않고 장바구니에 있는 모든 상품을 삭제 할 수 있습니다.

    4. 장바구니 담기

  • 장바구니 메뉴에서 상품 번호와 상품 수량을 입력하면, 장바구니에 상품을 등록할 수 있도록 개발했습니다.

    • 핵심코드

    1. 장바구니 리스트

    CartList Controller

  • 위 코드는 장바구니 리스트를 출력하고, 구매 여부를 확인하는 기능을 담당합니다
    • 장바구니 리스트 출력
  • CartListService를 실행하여 현재 록인한 회원의 장바구니 리스트를 가져옵니다.
  • 결과 리스트를 cartList 변수에 저장합니다.
  • CartPritn 클래스를 사용하여 장바구니 리스트를 출력합니다.
    • 장바구니 상태 확인
  • 장바구니가 비어 있을 경우, “장바구니가 비어 있습니다”라는 메시지를 출력합니다.
  • 장바구니에 상품이 있을 경우, 구매 여부를 묻습니다.
    • 구매 여부 확인
  • 사용자가 ‘y’를 선택하면
  • 현재 로그인한 회원의 번호를 가져옵니다.
  • 연락처, 배송지, 요청사항을 입력받습니다.
  • 입력받은 연락처, 배송지, 요청사항을 배열에 저장한 후 주문을 처리합니다.
